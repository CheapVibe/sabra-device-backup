Import Tips
General Guidelines
  • CSV files must be UTF-8 encoded
  • First row must contain column headers
  • Use pipe character (|) to separate multiple values in list fields
  • Boolean fields accept: Yes/No, True/False, 1/0
  • Empty cells use default values
Import Order for Dependencies
  1. Vendors - No dependencies
  2. Credentials - No dependencies (passwords set manually)
  3. Groups - No dependencies
  4. Devices - Requires credentials, groups
  5. Jobs - Requires devices, groups
