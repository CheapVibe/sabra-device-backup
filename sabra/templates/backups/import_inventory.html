Import Format

Import JSON files exported from Sabra Device Backup. The file should have been created using the Export Inventory feature.

What Gets Imported
  • Devices (name, hostname, IP, vendor, type, etc.)
  • Device Groups (if included in export)
  • Backup Jobs (if included in export)
  • Tags
What Doesn't Import
  • Passwords (security reasons)
  • SSH private keys
  • Configuration snapshots (use Config Import)
After Import

After importing devices, you'll need to:

  1. Create or update credential profiles with passwords
  2. Assign credential profiles to devices
  3. Test connectivity to devices
  4. Enable backup jobs
